MacBook AirにOllamaをインストール

 July 21, 2025に投稿  |    • 翻訳：en

はじめに

1ヶ月前、MacBook AirにLM Studioをインストールして、ローカルLLMを試す記事をアップした。今回は、OllamaでローカルLLMを試した。

関連情報

  1. M4 の Mac mini で ローカルLLM： Ollama で Gemma 3 を軽く試す - M4でローカルLLMを試す記事。
  2. Get up and running with large language models. - Ollamaの公式ページ。ここからダウンロード。
  3. ollamaを使ってローカル環境でLLMを動かす - DockerコンテナにOllamaを組み込んで、ubuntu環境でローカルLLMを動かすという、8ヶ月前の自分のブログ。あの頃は、RAGとナレッジグラフに取り組んでいた（ハマっていた）。
  4. Open WebUIをフロントエンドにしてollamaのLLMと会話する - 上記のブログの続きで、OllamaのコンテナのフロントにOpenWebUIを置いて、ChatGPTライクに使えるようにしたブログ。
  5. docker.desktop - Docker Desktopの公式ページ。こちらから、Mac用のインストーラーをダウンロード（Download for Mac - Apple Silicon）。

インストール

ダウンロード・インストール

関連情報2.のOllama公式ページから、macOS用のインストーラー（Ollama.dmg）をダウンロード。インストーラーを起動し、GUIでアプリケーション・フォルダに移動するとインストール完了。

LLMモデルのダウンロード・実行

情報源1.によると、ここでインストール後の画面面が表示されるとのことだったが、自分は、見逃したのか、気づかなかったのか、分からないが、ターミナルで以下を実行した。

% ollama run llama3.2

Ollamaがバックグラウンドで起動してた

後で、気付いたのだが、メニューバーにOllamaのアイコンが表示されていた。バックグラウンドでOllamaが起動されていた。OneDriveとBluetoothの間のアイコンを右クリックしたメニューから「Setting…」を選ぶと、次のような設定画面となる。

レスポンスは、llama3.2を同程度か若干早い印象。回答はシンプル（過ぎる！）。

12bモデルのgemma3をダウンロード・実行

MacBook Air M4は32GB搭載しているので、gemma3のもう少し大きなモデルも実行できそうなので、以下で12bモデルをダウンロード・実行した。

% ollama run gemma3:12b

同じ質問を投げたが、12bモデルも同じようにシンプルな回答。若干遅いかな！

MacでOpen WebUIを使う

Open WebUIを動かすために、Dockerが必要なので、先ずはDocker Desktopをインストールする。（Dockerを使わずにOpen WebUIを動かすやり方もあるようだ、ここを参考。またDocker Desktopの代替ツールについては、こちらを参考）

Docker Desktopをインストール

関連情報5.のDocker Desktop公式ページから、Apple Silicon用のインストーラ（Docker.dmg）をダウンロード。インストーラーを起動し、GUIでアプリケーション・フォルダに移動するとインストール完了。 「Welcome to Docker」画面で、Work/Personalかを聞かれ、パスワードを入力して設定が完了。

Docker Desktopは、正常に動作しているようだ！

open-webuiコンテナを起動

% docker run -d -p 3000:8080 --add-host=host.docker.internal:host-gateway -v open-webui:/app/backend/data --name open-webui --restart always ghcr.io/open-webui/open-webui:main

ブラウザからopen-webuiに接続

ブラウザで「http://localhost:3000/」と入力すると、Open WebUIに接続される。初回はメールアドレス、パスワードなどを入力する必要あり。適当に入れても大丈夫！

直接Ollamaプロンプトから質問した時よりも、最初に少し時間が掛かったように感じる。

OpenWebUIからもモデルの変更が出来る。

LLMの変更

左上の現在動作中のモデル名の右の[v]をクリックすると、使用できる（Ollamaでpullした）モデルの一覧が表示され、そこから切り替えられる。

まとめ

LM Studioの投稿でも述べたが、MacBook AirというラップトップでローカルLLMがそれなりに満足できる速度、回答内容で使えることは、素晴らしい。

研修カリキュラムの案を作成させたが、回答までの時間は、ChatGPTより遅いが、使えないレベルではない。回答内容についても、何度か質問を掘り下げていくと、満足できるレベルの回答を得ることができた。
